# Fresh Cart

Fresh Cart is a full-featured e-commerce web application built with **Next.js**, **React**, and **Tailwind CSS**.  
The application allows users to browse products, manage their cart and wishlist, and complete authentication flows with a modern and responsive UI.

## Features

- User Authentication (Sign up / Sign in / Logout)
- Protected Routes
- Browse Products
- View Product Details
- Add to Cart
- Remove from Cart
- Update Product Quantity
- Add to Wishlist
- Remove from Wishlist
- Real-time UI updates
- Loading states & error handling
- Toast notifications
- Fully Responsive Design

## Technologies Used

- Next.js
